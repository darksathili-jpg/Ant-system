# Ant-system
En 1992, un chercheur en informatique du nom de Marco Dorigo regardait des fourmis dans son jardin. Il avait placé deux chemins entre leur nid et de la nourriture : un court et un long. Après quelques minutes, presque toutes les fourmis empruntaient le chemin court — sans que personne ne leur ait dit lequel choisir.
